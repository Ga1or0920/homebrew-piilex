# homebrew-piilex

Homebrew tap for [piilex](https://github.com/piilex/piilex) -- PII Lexical Analyzer.

## Install

```bash
brew install piilex/tap/piilex
```

Or add the tap first:

```bash
brew tap piilex/tap
brew install piilex
```

## Upgrade

```bash
brew upgrade piilex
```

## Uninstall

```bash
brew uninstall piilex
```

## About

piilex detects personally identifiable information (PII) in TypeScript,
JavaScript, and Python source code, traces data flows, and maps findings
to GDPR and CCPA regulatory articles.

See the [main repository](https://github.com/piilex/piilex) for full documentation.
